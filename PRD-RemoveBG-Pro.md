# RemoveBG Pro - 产品需求文档 (PRD)

> **产品定位**：基于 Cloudflare + Remove.bg API 的零存储 AI 抠图工具
> **目标用户**：电商卖家、自媒体博主、需要快速抠图的普通用户
> **核心卖点**：无需登录、即用即走、零存储、全球 CDN 加速

---

## 一、需求分析

### 1.1 痛点挖掘

| 用户群体 | 核心痛点 | 现有解决方案缺陷 |
|----------|----------|------------------|
| **电商小卖家** | 产品图需要白底/透明底 | PS 学习成本高；外包贵（5-20元/张）；Remove.bg 免费额度少 |
| **自媒体博主** | 内容创作需要人物抠图换背景 | 手机 App 有水印；免费版功能受限；需要下载安装 |
| **HR/行政** | 员工证件照统一背景色 | 收集照片→发设计→等反馈，流程长 |
| **学生/个人** | 偶尔需要抠图（简历、社交） | 不想付费买软件；不想注册账号 |

### 1.2 竞品分析

| 竞品 | 定价 | 存储 | 登录要求 | 我们的差异化 |
|------|------|------|----------|--------------|
| **Remove.bg** | 免费50张/月，$0.20/张 | 云端存储 | 需要 | 更多免费额度，无需登录 |
| **PhotoRoom** | 订阅制 $5.99/月 | 云端存储 | 必须登录 | 纯免费工具，无付费墙 |
| **Remove.bg 国内仿品** | 看广告/付费 | 本地/云端 | 多数需要 | 无广告，界面简洁 |
| **Adobe Express** | 订阅制 | 云端存储 | 必须登录 | 轻量级，专注抠图 |

### 1.3 使用场景

```
【高频场景】
├── 电商 Listing 图制作（白底/透明底）
├── 小红书/朋友圈配图（换背景发内容）
├── 简历证件照（蓝底/白底切换）
└── PPT/设计素材提取

【中频场景】
├── 老照片修复（先抠图再处理）
├── 宠物照片处理
└── 产品宣传图制作

【长尾场景】
├── 表情包制作
├── 创意合成（换背景到任意场景）
└── 批量处理（后续版本）
```

## 四、技术方案

### 4.1 架构设计

```
用户浏览器
    ↓ 上传图片
Cloudflare Pages (静态前端)
    ↓ API 调用
Cloudflare Worker (后端代理)
    ↓ 转发请求
Remove.bg API
    ↓ 返回透明 PNG
Cloudflare Worker
    ↓ 返回结果
用户浏览器 (展示/下载)
```

### 4.2 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | HTML5 + TailwindCSS + Vanilla JS | 单页面应用，无框架依赖 |
| **后端** | Cloudflare Worker | 代理 API 请求，保护 Key |
| **API** | Remove.bg API | AI 抠图能力 |
| **部署** | Cloudflare Pages | 全球 CDN，免费额度充足 |
| **存储** | 无 | 图片仅存在于内存 |

### 4.3 关键设计决策

**为什么用 Worker 做代理？**
- ✅ 前端直接调用 Remove.bg 会暴露 API Key
- ✅ Worker 可以做请求限流、错误处理
- ✅ 后续可扩展缓存、日志等功能

**为什么零存储？**
- ✅ 降低架构复杂度
- ✅ 用户隐私保护（图片不落盘）
- ✅ 符合 Cloudflare 免费额度设计
- ✅ 与竞品形成差异化

## 五、部署方案

### 5.1 基础设施

| 服务 | 用途 | 费用 |
|------|------|------|
| **Cloudflare Pages** | 托管前端 | 免费 |
| **Cloudflare Worker** | API 代理 | 免费（10万次/天） |
| **Remove.bg API** | AI 抠图 | 免费50张/月 + $0.20/张 |
| **域名** | 可选自定义域名 | $10/年（可选） |

### 5.2 部署步骤

```
1. 创建 Cloudflare Worker
   └── 部署 worker.js，配置 REMOVE_BG_API_KEY

2. 创建 Cloudflare Pages 项目
   └── 连接 GitHub 仓库
   └── 配置构建设置：npm run build
   └── 输出目录：dist

3. 配置环境变量
   └── REMOVE_BG_API_KEY

4. 部署完成
   └── 获得 https://xxx.pages.dev 链接
```

### 5.3 免费额度评估

| 资源 | 免费额度 | 预计用量 | 是否足够 |
|------|----------|----------|----------|
| Worker 请求 | 10万次/天 | <1000次/天 | ✅ 充足 |
| Worker CPU | 50ms/请求 | <50ms | ✅ 充足 |
| Pages 请求 | 无限 | 低频 | ✅ 充足 |
| Remove.bg | 50张/月 | 50张/月 | ⚠️ 刚好 |

> **成本优化**：如需更多额度，可注册多个 Remove.bg 账号轮询使用。
